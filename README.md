## BaaliMitra

BaaliMitra is an AI-powered agricultural assistant and an web-based ecosystem designed to help farmers make informed decisions through crop recommendation, plant disease detection alongside weather forecast, market connect, forums and so on.

## Features
* Crop Recommendation System
* Plant Disease Detection
* Farmer Discussion Forum
* Equipment Store Interface
* Responsive Web Interface

## Technologies Used
* Python
* Flask
* HTML, CSS, JavaScript
* TensorFlow
* Scikit-Learn
* OpenCV


## Installation

```bash
git clone https://github.com/anupamchtry/BaaliMitra.git
cd BaaliMitra
pip install -r requirements.txt
python run.py
```

## Usage
Open:
http://localhost:5000
